# Changelog

All notable changes to this project will be documented here.

## v1.0.0

### Added

- Initial release of Afghan Typing Master
- Support for English, Pashto, and Dari languages
- Typing speed and accuracy testing
- Real-time statistics tracking
- Dark and light theme support
- Virtual keyboard display
- Sound effects for keystrokes and completion
- Lesson system for learning typing
- Word falling game mode
- Comprehensive settings customization
- Auto-update system for seamless updates

### Features

- Multi-language support (English, Pashto, Dari)
- Customizable test durations (15s, 30s, 1min, 2min)
- Live WPM and accuracy statistics
- Progress tracking and history
- RTL support for Pashto and Dari
- Responsive UI with modern design
- Command palette for quick navigation
- Onboarding for new users

### Technical

- Built with Electron, React, and TypeScript
- Uses electron-builder for packaging
- Auto-updates via GitHub Releases
- Secure IPC communication
- Local storage for user data
